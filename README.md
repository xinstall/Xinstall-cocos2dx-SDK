# Xinstall-cocos2dx-SDK
为了方便 cocos2d-x 集成 xinstall SDK 的二次封装

# Lua集成
使用`Lua`集成 Xinstall SDK 到 cocos2d-x 项目中，请使用 `Lua` 目录的文件，并参考[Lua 集成指南](https://github.com/xinstall/Xinstall-cocos2dx-SDK/blob/main/LuaSDK/README.md)

# JavaScript集成
使用`JS`集成 Xinstall SDK 到 cocos2d-x 项目中，请使用 `JavaScriptSDK` 目录的文件，并参考[JS 集成指南](https://github.com/xinstall/Xinstall-cocos2dx-SDK/blob/main/JavaScriptSDK/README.md)
